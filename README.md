# DSA Practice in JavaScript

This repository contains Data Structures and Algorithms (DSA) practice problems implemented in JavaScript.

## Structure

- **Array/**: Array-related problems and solutions
- **Recursion/**: Recursion-based problems and solutions

## How to Run

Each problem has its own JavaScript file. You can run them using Node.js:

```bash
node Array/search.js
```
